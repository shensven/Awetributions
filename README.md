## FEATURES

-   Multiple Languages
-   Dark mode
-   Debounce & Throttle
-   High Concurrency

## PREREQUISITES

-   Node 12 or newer
-   Watchman
-   JDK 8 or newer
-   Android minSDK 21
-   Xcode
-   CocoaPods

## BUILD

### Run instructions for Android

Have an Android emulator running (quickest way to get started), or a device connected.

```sh
yarn android
```

### Run instructions for iOS

```sh
yarn ios
```

### Build for Android

```sh
cd android && ./gradlew assembleRelease
```
